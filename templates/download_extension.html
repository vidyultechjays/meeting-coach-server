Installation Guide

1

Download the Extension

Click the download button at the top of this page to get the extension ZIP file

2

Extract the ZIP File

Right-click the downloaded ZIP file and select "Extract All..." option

3

Open Chrome Extensions

Go to Chrome and select Extensions → Manage extensions from the side panel

4

Enable Developer Mode

Turn on the Developer mode toggle button in the top right corner

5

Load the Extension

Click on "Load unpacked" button and select the extracted folder of the extension

6

Use in Google Meet

When you enter a Google Meet, select the extension from the extensions icon in the top right panel
